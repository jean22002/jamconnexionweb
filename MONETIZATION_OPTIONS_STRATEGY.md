## 📈 MÉTRIQUES À SUIVRE

### KPIs Essentiels

1. **Taux de conversion** :
   - Gratuit → Payant (objectif : 5-10%)

2. **LTV (Lifetime Value)** :
   - Durée moyenne abonnement × prix
   - Objectif : > 6 mois

3. **Churn Rate** :
   - % désabonnements/mois
   - Objectif : < 5%

4. **ARPU (Average Revenue Per User)** :
   - Revenus totaux / Utilisateurs actifs
   - Objectif : 2-3€/utilisateur/mois

5. **CAC (Coût d'Acquisition Client)** :
   - Dépenses marketing / Nouveaux clients
   - Objectif : CAC < LTV/3
